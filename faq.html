FAQ

Hoe kan ik een document printen vanaf mijn laptop op de HZ?


Je kan digitaal verbinden met de printers op HZ, je kunt dus vanaf overal waar je verbinding hebt met het netwerk printen. Stuur het bestand naar de printer, en houd je studentenpas voor de printer. Nu kun je printen, houd er wel rekeningen mee dat de pas opgeladen moet zijn en dit geld kost.

Hoe kan ik een document inscannen en het naar mijn laptop bij de HZ sturen?


Dit kan via het kopieer apparaat, zorg er wel voor dat jouw laptop verbonden is met het netwerk en de printer.

Hoe kan ik iets kopen (zoals wanner ik mezelf inschrijf voor het IT introductie event op vrijdag 16/09) op de HZ web shop?


Dit kan via de HZ shop die hier te vinden is.

Waar kan ik de nieuwste instructies vinden over wat je moet doen als je ziek bent/ Covid symptonen? vertoont?


De instructies bij ziekte kun je op dit gedeelte van de HZ site vinden.

Hoe kan ik een project ruimte boeken in een van de vleugels van het gebouw?


Ga naar jouw learn pagina van de HZ site, en navigeer vanuit hier naar het Selfservice portaal. Vanuit hier kan je reserveren.

Wat zijn de instructies als ik mijn auto wil parkeren op de HZ parkeerplaats?


Je kunt het parkeerterrein betreden als je jouw HZ pas toont bij de slagboom, kom je als gast kun je hier aanbellen. Na het passeren van deze slagboom volg je de borden naar de HZ parkeerplaatsen deze zijn gemarkeerd met witte HZ bordjes.
